# Greenden-Tailwind

Responsive flower shop landing page built with HTML & Tailwind CSS v4. Features sticky nav, best sellers grid, customer reviews, newsletter signup & contact form. Mobile-first design with hover effects. Perfect starter project! A modern, responsive flower shop website built with HTML5 & Tailwind CSS v4. Perfect blend of clean design and smooth interactions.

Features
Sticky responsive navbar with mobile hamburger menu

Hero section with CTA buttons and plant stats

Best sellers grid (Rose, Money, Pine, Snake plants)

Customer reviews carousel-style cards

Contact form with social media links

Newsletter signup

Hover animations and mobile-first design

Quick Start
bash npm install npx tailwindcss -i ./input.css -o ./output.css --watch Link output.css in your HTML and you're live!
